ELASTICSEARCH

Elasticsearch is a search engine based on the Lucene library. It provides a distributed, multitenant-capable full-text search engine with an HTTP web interface and schema-free JSON documents.

Elasticsearch is a distributed, RESTful search and analytics engine capable of addressing a growing number of use cases.

STEPS

  1. Install and modify elasticsearch.yml file if needed
  2. Start or restart service
sudo apt-get elasticsearch sudo systemctl restart elasticsearch
LOGSTASH

Logstash is an open-source data collection, parsing, and enrichment pipeline that helps you ingest data from various sources, transform it, and ship it to your preferred destination, such as Elasticsearch.

Logstash dynamically ingests, transforms, and ships your data regardless of format or complexity.

STEPS

  1. Install and create a configuration file to be stored in configs. This will specify how input and output is parsed
  2. Start or restart service
sudo apt-get logstash sudo systemctl restart logstash
KIBANA

Kibana is a source-available data visualization dashboard software for Elasticsearch.

Kibana provides visualization capabilities on top of the content indexed on an Elasticsearch cluster.

Users can create bar, line and scatter plots, or pie charts and maps on top of large volumes of data.

STEPS

  1. Install and run
sudo apt-get kibana sudo systemctl start kibana
CONCLUSION

The ELK Stack is a collection of three open-source projects: Elasticsearch, Logstash, and Kibana.Elasticsearch is a full-text search and analytics engine, while Logstash is a log aggregator that collects and processes data from multiple sources, converts, and ships it to various destinations, such as Elasticsearch.

It enables users to take data from any source, in any format, and search, analyze, and visualize it in real-time.

The ELK Stack was formerly known as the ELK Stack for the core products in the group, but has been rebranded as the Elastic Stack.

AWS offers various ingestion tools to support the ELK Stack
